# python-Random-Password-generator
 #Project Name:
Random Password Generator
#Objective:
      To create a Python program that generates a random and secure password based on the length entered by the user.
#Concepts Used:
    Random Module (random)
    String Module (string)
    User Input (input())
    Loops (for)
    String Manipulation
#How It Works:
  Ask the user to enter the lemgth of the password.
  Store letters and numbers in a character set.
  Randomly select characters from the set.
  Repeat until the required password length is reached.
  Display the generated password to the user.
